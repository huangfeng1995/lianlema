# 练了吗 - App Store 素材准备指南

> 本文档列出在 App Store 上架「练了吗」App 所需的所有素材规格和要求。

---

## 1. App 图标 (App Icon)

App Store 要求上传 **1024×1024 像素**的 PNG 图片（不带圆角和阴影）。

### 各平台尺寸要求

| 平台 | 尺寸 | 用途 |
|------|------|------|
| App Store | 1024×1024 px | 上架主图（必传） |
| iPhone | 180×180 px (60pt@3x) | iPhone 主屏幕 |
| iPhone | 120×120 px (60pt@2x) | iPhone 主屏幕（老设备） |
| iPad | 167×167 px (83.5pt@2x) | iPad 主屏幕 |
| iPad | 152×152 px (76pt@2x) | iPad 主屏幕（老设备） |
| App Store | 1024×1024 px | App Store 搜索展示 |

### 设计建议

- **背景色**：使用 `#F5F0EB`（宣纸米白）或 `#E85A1C`（品牌橙）渐变
- **图形元素**：中文「练」字或抽象火焰/打卡图形
- **风格**：简约、现代、扁平化
- **四角**：App Store 图标无需处理圆角，系统自动生成

## 2. App Store 截图 (Screenshots)

截图必须为真实设备截图，不可使用模拟器或设计图。

### iPhone 屏幕尺寸

| 设备 | 尺寸（像素）| 比例 |
|------|-------------|------|
| iPhone 16 Pro Max | 1290×2796 | 6.7" |
| iPhone 16 Pro / 15 Pro | 1179×2556 | 6.1" |
| iPhone 16 / 15 | 1179×2556 | 6.1" |
| iPhone 14 Pro Max | 1290×2796 | 6.7" |
| iPhone 14 Pro | 1179×2556 | 6.1" |
| iPhone SE (3rd) | 750×1334 | 4.7" |

### iPad 屏幕尺寸

| 设备 | 尺寸（像素）| 比例 |
|------|-------------|------|
| iPad Pro 12.9" | 2048×2732 | 6th Gen |
| iPad Pro 11" | 1668×2388 | 4th Gen |
| iPad Air 10.9" | 1640×2360 | 5th Gen |

### 截图要求

- **格式**：PNG 或 JPEG
- **颜色**：sRGB
- **无缩放**：像素对像素显示
- **可上传最多 10 张**（不同尺寸需分别上传）
- **优先尺寸**：6.7 寸和 6.5 寸截图放在最前面

### 截图套壳建议

可使用 [AppLaunchpad](https://www.applaunchpad.io)、[Mockuuups Studio](https://mockuuups.studio) 等工具自动生成设备模型展示图。

---

## 3. App 预览视频 (App Preview)

- **时长**：15-30 秒
- **格式**：H.264 编码的 MP4，最大 500MB
- **分辨率**：与截图尺寸一致
- **内容**：展示核心功能（打卡、目标设定、成就系统）

---

## 4. 文案内容

### 一句话简介（最多 30 字符）

```
每天行动，成为想成为的人
```

或

```
职场人的习惯追踪助手
```

### 详细描述（最多 4000 字符）

```
🔥 练了吗 - 每天行动，成为想成为的人

「练了吗」是一款帮助职场人每天行动的习惯追踪App。

【核心理念】
我们相信，真正改变人生的不是宏大的目标，而是每天微小的行动。通过「反愿景」锁定你最不想成为的人，用「每日杠杆」撬动人生改变。

【核心功能】

📋 反愿景设定
写下你最不想成为的人的样子，锁定后一年不可更改。每天看到它，时刻警醒自己。

🎯 每日杠杆
每天设定3个核心行动（杠杆），完成即可打卡。简单高效，不贪多。

🔥 连续打卡
坚持连续打卡，积累你的火焰天数。每一次打卡都是对自我的承诺。

🏆 月度Boss战
每月设定一个挑战目标，30天HP制，用打卡天数积累HP值。

⭐ 等级与成就
完成任务获得XP，升级解锁新等级称号。从「职场新人」到「人生赢家」。

【特色】

☀️ 极简模式
一键切换，隐藏动画特效，专注内容本身。

⏰ 每日提醒
自定义提醒时间，帮助你养成每日行动的习惯。

📊 打卡日历
直观的日历视图，回顾每一天的行动轨迹。

🎨 温暖设计
宣纸米白背景，炭火橙品牌色，给你不一样的温暖体验。

【适合人群】

• 想要养成好习惯的职场人
• 希望每天都有进步的你
• 想要摆脱焦虑、付诸行动的你
• 追求自我成长的长期主义者

开始行动，从今天开始。

「练了吗」- 每天行动，成为想成为的人。
```

### 关键词（最多 100 字符）

```
习惯追踪,每日打卡,自律,目标管理,行动力,职场成长,习惯养成,效率,自我提升,每日计划
```
