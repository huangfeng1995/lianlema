## 二、宠物外观系统

### 外观阶段（通过打卡解锁）

每只宠物有5个外观阶段：

| 阶段 | 解锁条件 | 外观描述 |
|-----|---------|---------|
| Lv1 初始 | 获得宠物时 | 基础形态，略显稚嫩 |
| Lv2 成长 | 累计打卡14天 | 开始发育，有些变化 |
| Lv3 进化 | 连续打卡30天 | 明显进化，更强更美 |
| Lv4 绽放 | 连续打卡60天 | 完全绽放，光彩夺目 |
| Lv5 传说 | 连续打卡100天 | 传说形态，独一无二 |

### 外观改变类型
- **颜色变体**：基础色 + 5种渐变/限定色
- **装饰配件**：帽子、眼镜、围巾、披风等
- **特效光效**：光环、粒子、火焰/水滴等元素特效

## 五、宠物心情系统升级

### 心情值（0-100）
| 心情值 | 状态 | 描述 |
|-------|------|------|
| 80-100 | 超开心 😄 | 宠物特别活跃，有特殊对话 |
| 60-79 | 开心 😊 | 正常积极状态 |
| 40-59 | 平静 😐 | 日常等待 |
| 20-39 | 有点丧 😢 | 打卡少，需要关注 |
| 0-19 | 很低落 😞 | 长期懈怠，需要激励 |

### 心情影响
- 心情高时，宠物更主动发消息鼓励
- 心情低时，宠物会更多询问"今天怎么了"
- 心情归0时，宠物进入「休眠」状态，需要喂食+连续3天打卡才能恢复

### 心情恢复方式
- 喂零食：+1~+5（根据零食类型）
- 打卡：+3
- 和宠物说话：+1
- 连续打卡：额外+2

## 六、技术实现要点

### 数据模型扩展
```dart
class PetV2 {
  final String typeId;           // 宠物类型ID (1-15)
  final String name;             // 名字
  final int appearanceLevel;      // 外观等级 (1-5)
  final int mood;                // 心情值 (0-100)
  final int currency;            // 宠物币
  final List<String> ownedItems; // 已拥有的物品ID列表
  final String equippedCostume;  // 当前穿戴的外观
  final List<String> equippedDecorations; // 当前家居装饰
  final DateTime createdAt;
  final DateTime lastFedAt;
  final int totalCheckIns;        // 累计打卡天数
}

class ShopItem {
  final String id;
  final String name;
  final String description;
  final int price;
  final String category; // 'costume' | 'snack' | 'decoration'
  final String icon;
  final int effect; // 零食的心情恢复值
}
```

### 宠物类型配置
```dart
const petTypes = [
  PetType(id: 'fire_1', name: '炭炭', element: 'fire', personality: 'warm', probability: 0.20, ...),
  PetType(id: 'fire_2', name: '闪焰', element: 'fire', personality: 'driven', probability: 0.08, ...),
  // ... 15种
];
```

### 随机分配算法
```dart
PetType assignRandomPet() {
  final random = Random();
  final r = random.nextDouble(); // 0.0 ~ 1.0
  double cumulative = 0;
  for (final pet in petTypes) {
    cumulative += pet.probability;
    if (r <= cumulative) return pet;
  }
  return petTypes.first; // fallback
}
```

## 九、界面流程

### 新用户领养流程
```
开机 → Splash → 填写完 onboarding → 显示"正在为你匹配宠物..." → 
动画：一只宠物从光中走来 → "你的宠物伙伴是：炭炭！" → 
介绍宠物的特点 → 进入主页
```

### 宠物家入口流程
```
点击TabBar「宠物家」→ 进入宠物家界面 → 
可选择：查看宠物 / 喂零食 / 换装 / 装饰 / 互动
```

### 商店购物流程
```
点击商店 → 选择分类（外观/零食/家居）→ 
点击商品 → 确认价格和效果 → 确认购买 → 
宠物币扣除 → 物品加入背包/立即生效
```
