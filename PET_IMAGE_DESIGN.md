# 宠物形象设计需求

## 产品概述

「练了吗」是一款帮助职场人养成习惯的打卡 App，宠物系统是核心情感化设计。用户通过每日打卡养成宠物，宠物会成长、进化，给用户带来陪伴感和成就感。

品牌调性：温暖、有力量、真实成长

## 外观等级系统（5级）

每只宠物有5个外观阶段，通过累计打卡天数解锁：

| 等级 | 名称 | 解锁条件 | 视觉特征 |
|-----|------|---------|---------|
| Lv1 | 初始形态 | 默认 | 基础形象，较小 |
| Lv2 | 成长形态 | 累计14天 | 长大一圈，细节更丰富 |
| Lv3 | 进化形态 | 连续30天 | 色彩更鲜艳，有光效 |
| Lv4 | 绽放形态 | 连续60天 | 特效增强，周围有粒子 |
| Lv5 | 传说形态 | 连续100天 | 全身发光，有特殊光效 |

## 设计风格要求

### 整体风格
- **温暖真实**：避免过度卡通，接近插画质感
- **有性格**：每种宠物有独特的表情和姿态
- **可成长**：同一种宠物不同等级要有明显视觉差异
- **炭火主题**：配色以暖橙 (#FF6B35) + 深炭黑 (#1A1A1A) 为主

### 表情系统
每个宠物需要以下情绪状态的表情：
1. **开心/超开心** — 打卡成功时
2. **平静** — 日常状态
3. **有点丧** — 心情低落时
4. **很低落** — 心情极低时
5. **惊讶** — 收到礼物/新成就时

### 动作/动画
- **待机动画**：轻微的呼吸/摇摆循环（2-3秒）
- **开心动画**：原地小跳或摇尾巴（1-2秒）
- **庆祝动画**：收到礼物或升级时（2-3秒）

### 输出要求
- 格式：PNG透明背景 或 SVG
- 尺寸：建议 512×512px 起（适配不同分辨率）
- 每个宠物每个等级每个表情：5状态 × 5等级 × 15宠物 = **375张**
- 实际优先级：先做 Lv1（基础形态），其他等级逐步做

### 文件命名规范
```
assets/images/pets/{type}/{type}_lv{lv}_{mood}.png
示例：
assets/images/pets/fox/fox_lv1_happy.png
assets/images/pets/rabbit/rabbit_lv3_excited.png
```
